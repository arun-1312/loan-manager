A practical, offline-first Flutter application designed to track loans, dues, and payments in a clean, structured way. 

##  Why this exists

In many small-scale setups, loan tracking is still handled via fragmented, unreliable methods:
-  Lost notebooks
-  Buried WhatsApp messages
-  Pure memory 

This app replaces that chaos with a structured, automated system that is actually practical for daily use—no complicated onboarding, and absolutely no internet connectivity required.

## Core Features

- **End-to-End Loan Management:** Create and track loans with principal and interest calculations.
- **Smart Due Tracking:** Features a fixed monthly due system that automatically calculates pending dues based on time passed.
- **Payment Recording:** Seamlessly log payments (interest + principal) to instantly update balances.
- **Borrower Profiles:** Store critical details (name, location, phone number) with tap-to-call functionality.
- **Status Flagging:** Automatically highlights overdue loans so nothing slips through the cracks.
- **Instant Search:** Find borrowers and records instantly.
- **100% Offline:** Fully functional without an internet connection.

## Tech Stack

- **Framework:** Flutter / Dart
- **Storage:** SQLite (Local, offline-first database)
- **Design:** Material UI
